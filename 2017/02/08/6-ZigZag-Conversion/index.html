6. ZigZag Conversion

This question is a little tricky. There are 2 ways to solve it. First of all, you find calculate the law the ZigZag in different number of row. Secondly, you can use a 2D StringBuilder to store every character in ZigZag position. And then, you can connect the 2D StringBuilder together. So it is the final result.

###The Question:

The string “PAYPALISHIRING” is written in a zigzag pattern on a given number of rows like this: (you may want to display this pattern in a fixed font for better legibility)

P   A   H   N
P L S I I G
Y   I   R

And then read line by line: “PAHNAPLSIIGYIR”
Write the code that will take a string and make this conversion given a number of rows:

string convert(string text, int nRows);

convert(“PAYPALISHIRING”, 3) should return “PAHNAPLSIIGYIR”.

In my point of view, I prefer the second solution. Because when you are in an interview, you will be nervous that you cannot calculate the right law of ZigZag. So the 2D StringBuilder will be easy to understand and it has O(n) complexity.

The code part

public class Solution {
    public String convert(String s, int numRows) { 
    // try to find the law of ZigZag
    // corner condition when s == 0;
    if (s.length() < 1 || numRows <= 1) return s;
        StringBuilder[] sb = new StringBuilder[numRows];
    for (int i = 0; i < numRows; i++) {
        sb[i] = new StringBuilder("");
    }
    int index = 0;
    int incre = 1;
    for (int i = 0; i < s.length(); i++) {
        sb[index].append(s.charAt(i));
        if (index == 0) incre = 1;
        if (index == numRows - 1) incre = -1;
        index += incre;
    }
    for (int i = 1; i < numRows; i++) {
        sb[0].append(sb[i]);
    }
    return sb[0].toString();
    }
}
